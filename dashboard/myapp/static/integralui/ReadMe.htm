IntegralUI Studio for Web

v3.0

IntegralUI Studio for Web is a suite of advanced AngularJS directives and jQuery widgets. All components have many features like: fully customizable appearance, optimized for fast data loading and update, improved drag and drop functionality, animations, support for right-to-left layouts, themes and much more.

What's included

  • Accordion - A list of expandable groups with header and content panel
  • CheckBox - Native AngularJS directive that represents a check box
  • ComboBox - Native AngularJS directive that represents a combo box
  • ContextMenu - Adds a custom context menu to other AngularJS Directives
  • Grid - Fast Grid component that displays tabular data sets
  • Menu - Allows you to easily navigate among pages in your web apps
  • SlideBar - Animates slides composed of images or custom content
  • TabStrip - Allows you to create tabbed user interface in your web apps
  • TreeGrid - A Grid component that shows hierarchical data structures in multiple columns
  • TreeView - Fast data loading and update of hierarchical structures

What's New

  • New directives:
    • Grid - native AngularJS directive that displays tabular data sets
    • SlideBar - native AngularJS directive that animates slides composed of custom content
  • Templates - allows you to add custom content to each cell and item in Grid, TreeGrid and TreeView directives
  • Grid grouping - option to dynamically create group fields in Grid directive
  • Pagination - splits the large data sets in multiple pages
  • Touch support - scrolling, drag-drop, selection using touch
  • Option to change the appearance of expand boxes on hover
  • Show/hide of expand box on hover with animation
  • Option to select multiple rows and items from code
  • New properties:
    • grouping, an object that determines whether grouping feature is enabled in Grid directive
    • paging, an object that determines whether pagination feature is enabled and the page size
    • showExpandBoxes, determines whether expand boxes are visible or not in TreeView directive
    • templateSettings, an object that specifies how template is used in TreeView directive
  • New methods:
    • clearSelection, removes the selection status from tabs in TabStrip directive
    • getItemAt, which allows you to get the target item using mouse position
    • getRowAt and getCellAt, which allows you to get the target row and cell using mouse position
    • selectedItems, allows you to select multiple items from code
    • selectedRows, allows you to select multiple rows from code
  • New events:
    • dragOver and dragDrop events now include the targetCell field
  • and much more ...

What's Fixed:

  • Vertical Scrollbar is no longer shown when all columns are hidden
  • The order of moving column is now correctly set
  • When tabs are removed from close button, the scroll position is now updated
  • Scrolling is suppressed when loading animation is active
  • When checkbox is disabled the checkmark is grayed
  • Setting the showHeader property value to false, hides the grid header while still showing the rows
  • CSS class for linear loading animation is now available for customization
  • Fixed the problem which prevents cancelation of drag drop over control empty space
  • Fixed an issues which causes changes to the referencing data object when data is loaded from a remote data source
  • During dynamic resize, the size of header and footer block element is correctly updated
  • The maximum scroll position is now correctly set to default when there is no scrollbar visible
  • Multi selection now functions correctly when view is scrolled and CTRL key is not pressed
  • Multiple selected rows are cleared on right-click
  • Context menu applied to the grid no longer prevents firing of cellClick and rowClick events
  • Grid cells with image no longer throws an error when image URL is not retrieved on time during data binding
  • Other minor bug fixes
